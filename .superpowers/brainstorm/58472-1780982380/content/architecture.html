技术栈

前端 React + TypeScript + Vite + Tailwind CSS
后端 Python FastAPI + Uvicorn
音高模型 CREPE (MIT License)
节奏分析 Librosa (ISC License)
人脸检测 OpenCV Haar Cascade / MediaPipe
数据库 SQLite（MVP）→ 后续可升 PostgreSQL

数据流

  1. 学生打开浏览器 → 摄像头自动开启（人脸检测确认身份）
  2. 选择考试曲目 → 显示参考旋律 → 开始录音
  3. 音频数据流传到后端 → CREPE 分析音高 + Librosa 分析节奏
  4. 后端返回实时评分结果 → 前端渲染音高曲线 + 节奏对比 + 分数
  5. 演唱结束 → 生成最终评分报告 → 存入数据库
