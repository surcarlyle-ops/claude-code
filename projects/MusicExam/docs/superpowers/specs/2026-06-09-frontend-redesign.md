# MusicExam 前端界面美化设计方案

> 版本：v1.0 | 日期：2026-06-09 | 状态：待实施

## 设计定位

**温暖马卡龙治愈系** — 面向初高中生的音乐考试练习工具，低AI味、柔和自然、像人工设计的温柔App。

## 新增页面

### 首页 `/home`（新增）
- 顶部："Hello, 学生名"标题 + 圆形头像
- 个人信息标签：3个并排小标签（性别、年级、上次得分），白底圆角灰边框
- 今日进度：双卡并排（粉色=演唱练习进度带进度条数字，黄色=已完成曲目数）
- 快捷功能：一排彩色圆按钮（演唱练习/视唱练耳/错题回顾/历史记录/+）
- 待办任务：垂直彩色卡片列表（不同颜色区分，带时间/任务名/说明/勾选框）

### 趋势页 `/trends`（新增）
- 顶部：返回箭头 + 标题 + 时间筛选（7天/30天/自定义）
- Canvas 折线图：得分趋势可视化，柔和浅色填充
- 练习记录列表：按日期排列的评分卡片

## 美化页面

### 登录页 `/login`
- 按马卡龙风格调整配色、圆角、留白
- 保留演示模式按钮（视觉统一后）

### 选曲页 `/songs`
- 彩色曲目卡片按统一圆角风格
- 筛选标签圆润化

### 考试页 `/exam`
- 分数卡三张（蓝/粉/绿）按马卡龙色系重做
- 音高曲线 Canvas + 简谱滚动 + 进度条统一样式

### 结果页 `/result`
- 总分大卡 + 分项评分卡片 + 错误标记 + 建议卡片
- 按新配色方案统一样式

## 配色
主色：浅粉 `#FADADD`、奶油黄 `#FDF4D0`、浅天蓝 `#D6ECFB`、薄荷绿 `#D5F5E3`、淡紫 `#E8DAEF`
点缀：珊瑚粉 `#FF6B6B`、青蓝 `#4ECDC4`
底色：暖白 `#FFFAF5`

## 数据策略
- 首页待办任务、个人信息标签：静态写死
- 快捷功能未实现项：占位+提示"即将上线"
- 趋势页图表数据：静态示例数据
