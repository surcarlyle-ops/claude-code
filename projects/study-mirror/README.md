# 🎯 学迹 LearnWitness - 你的学习见证者

> **一个为自己记录学习旅程的温和见证者，偶尔也能帮你向他人优雅地展示这段旅程。**

---

## 🌈 **项目理念**

**核心理念**: 「学习镜子」→ 为自己成长记录，而非取悦导师

**产品灵魂**：温和、温暖、鼓励式的个人学习伙伴

---

## 🏗️ **技术架构**

### **MVP 版本 (已完成)**
1. **侧边栏极简记录组件** - 一句话+标签，随时随地记录
2. **计时器组件（带简单仪式感）** - 开始/停止学习，温暖提示
3. **学习时间热图组件** - GitHub贡献图风格，可视化努力密度
4. **纯本地数据存储** - localStorage，完全私密
5. **温暖亲切的UI界面** - 暖色系，温柔语气，鼓励氛围

### **技术选择**
- **前端**: React + TypeScript + TailwindCSS
- **数据存储**: localStorage/IndexedDB
- **架构**: 单机离线优先，完全不需要后端
- **设计**: 温暖渐变色系，圆角设计，柔和阴影

## 🚀 **如何使用**

### **快速开始**
```bash
cd frontend
# 如有npm依赖问题，可以直接打开index.html
# 这是一个纯静态页面，不需要编译
```

### **核心功能体验**

#### **1. 侧边栏极简记录**
- 右侧边栏常驻，随时可以记录
- 一句话内容 + 标签 + 感觉选择
- 自动添加时间相关标签
- 保存即显示成功反馈

#### **2. 计时器学习**
- 点击「开始今天的学习」开始计时
- 学习过程中显示专注时间
- 结束时填写学习收获和感觉
- 自动生成学习统计

#### **3. 学习热图**
- 类似GitHub贡献图的视觉展示
- 6个月学习密度可视化
- 颜色深浅反映投入强度
- 点击查看每天详细数据

#### **4. 数据隐私**
- **所有数据保存在本地**，不上传任何服务器
- 支持数据导出/导入 (JSON格式)
- 随时可以清空数据
- 真正的「自我学习见证」

## 🎨 **设计特色**

### **视觉风格**
- **色彩**: 温暖的咖啡色系 + 平静的青色系
- **字体**: Inter (无衬线) + Merriweather (衬线)
- **圆角**: 温和的圆角设计，避免尖锐
- **阴影**: 轻柔的内阴影和外阴影
- **动画**: 温和的呼吸动画和浮动效果

### **交互体验**
- **亲切文案**: 「早安！新的一天从学习开始」
- **温柔语气**: 不催促，不评价，只陪伴
- **即时反馈**: 记录成功、计时完成都有温暖反馈
- **低压力**: 不需要详细规划，随笔记录即可

### **数据隐私设计**
- **纯本地**: 数据只存在浏览器中
- **无网络**: 完全离线运行
- **可选分享**: 如果要给导师看，手动导出/截图
- **随时可丢弃**: 清空按钮就在手边

## 🤔 **关于这个项目的思考**

### **为什么选择纯前端实现？**
1. **隐私保护** - 数据永远在本地，不涉及任何隐私担忧
2. **零门槛** - 不需要部署后端，任何人都能立即使用
3. **个人化** - 工具完全为你所用，不会因为商业化而变质
4. **简单纯粹** - 不引入复杂概念，专注核心价值

### **与其他学习工具的区别**
| 功能 | Notion | LearnWitness (本项目) |
|------|---------|------------------|
| 学习记录 | 万能但空白 | 引导式，低门槛 |
| 学习计时 | 需要手动 | 内置计时器，有仪式感 |
| 进度可视化 | 依赖自己搭建 | 自动生成热图 |
| 隐私性 | 云端存储 | 纯本地，不上传 |
| 使用动机 | 需要规划 | 记录本身就是激励 |

### **项目的哲学**
> 学习不是一场表演。
> 
> 真正的成长发生在安静的时刻，发生在自己与知识的对话中。
> 
> LearnWitness不是要让你「看起来在学习」，
> 
> 而是要帮你「真实地看见自己的学习」。

## 💡 **给用户的建议**

如果你决定使用这个工具：

### **使用习惯建议**
1. **每天只记录10秒钟** - 不求完美，只求持续
2. **标签自由发挥** - 想到什么就标什么
3. **不看别人的热图** - 只看自己的成长曲线
4. **接受不完美** - 空白的日子也很美

### **心理建设**
- 这个工具不是为了「证明」学习
- 这个工具是为了「感受」学习
- 数据是为你服务，不是你为数据服务
- 成长是过程，不是产品
